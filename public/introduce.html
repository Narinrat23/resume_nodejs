สวัสดีค่ะ ดิฉัน นางสาวนลินรัตน์ คัมภีร์วโรดม ชื่อเล่น นาเดีย ปัจจุบันอายุ 22 ปี
เกิดวันที่ 23 พฤศจิกายน พ.ศ. 2545 สัญชาติ ไทย
ปัจจุบันกำลังศึกษาอยู่ชั้นปีที่ 4 คณะวิศวกรรมศาสตร์ สาขาอิเล็กทรอนิกส์และระบบคอมพิวเตอร์
มหาวิทยาลัยศิลปากร วิทยาเขตพระราชวังสนามจันทร์ โดยดิฉันมีข้อดี ข้อเสียดังนี้

จุดเด่น grade

  • มีความตั้งใจในการทำงานที่ได้รับมอบหมายเป็นอย่างดี

  • สามารถทำงานเป็นทีมและเข้ากับผู้อื่นได้อย่างเหมาะสม

  • สามารถแก้ปัญหาเฉพาะหน้าได้อย่างรวดเร็ว

จุดด้อย grade

  • กดดันตัวเองมากเกินไป

  • ทำงานเร็วอาจมีข้อผิดพลาดเกิดขึ้นได้

  • ทางด้านภาษา ภาษาอังกฤษยังไม่คล่องแคล่วมากนัก

      ซึ่งข้อเสียที่กล่าวมานั้น ดิฉันกำลังปรับปรุงเพื่อให้สามารถทำงานร่วมกับผู้อื่นได้เป็นอย่างดี
และยังปรับปรุงเพื่อให้เป็นคนที่ดีขึ้นไม่ว่าจะเป็นในเรื่องของการทำงานหรือการใช้ชีวิตประจำวัน
      โดยดิฉันยังมีทักษะเด่นที่น่าสนใจสำหรับตัวเอง ไม่ว่าจะเป็น Soft skill หรือ Hard skill ดังนี้

Soft Skill grade

  • ทักษะการสื่อสาร (Communication)
    • สามารถสื่อสารกับผู้อื่นได้เป็นอย่างดีและมีความเหมาะสม

  • ทักษะการบริหารเวลา (Time Management)
    • สามารถบริหารการจัดการเวลากับงานที่ได้รับมอบหมายได้

  • ทักษะการทำงานร่วมกับผู้อื่น (Collaboration)
    และทักษะการปรับตัว (Adaptability)
    • สามารถทำงานร่วมกับผู้อื่นพร้อมกับปรับตัวเข้ากับสังคมได้เป็นอย่างดี

  • ทักษะการตัดสินใจ (Decision Making)
    • สามารถตัดสินใจอย่างมีเหตุผลและสามารถแก้ปัญหาเฉพาะหน้าได้

  • ทักษะการมีทัศนคติเชิงบวก (Positive Attitude)
    • สามารถเป็นพลังบวกสำหรับคนรอบข้างได้เป็นอย่างดี

Hard Skill grade

  • ทักษะด้านคอมพิวเตอร์และการใช้งานโปรแกรม (Computer Software and Application Knowledge)
       สามารถใช้โปรแกรมได้หลายประเภท

    • Microsoft 365

    • Python, C++

    • Java script

    • LabView

    • ISIS7 Professional

  • ทักษะด้านการออกแบบ (Design)
       สามารถออกแบบโปรเตอร์หรือการออกแบบเว็บไซต์ในการทำงานโดยใช้โปรแกรมดังนี้ได้

    • Visual Studio code

    • Canva

โดยทักษะต่างๆดิฉันยังพัฒนาและฝึกฝนตนเองอยู่ตลอดในการใช้งาน
อีกทั้งยังมีโปรเจคที่เคยเข้าร่วมในการพัฒนาการฝึกฝนทั้งทักษะด้าน Soft skill และ Hard skill ดังนี้
